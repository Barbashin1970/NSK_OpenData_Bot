# СИГМА — Руководство пользователя

> **Умный помощник по открытым данным Новосибирска**  
> Центр искусственного интеллекта НГУ · v1.1.0  
> Веб-интерфейс: [nsk-opendata-bot.up.railway.app](https://nsk-opendata-bot.up.railway.app/)

---

## Что такое СИГМА

СИГМА — это веб-бот, который отвечает на вопросы о городе **на живом русском языке**. Никаких меню, кнопок и выпадающих списков — просто пишите или говорите запрос, бот отвечает. Данные берутся напрямую из официальных источников мэрии, Open-Meteo, OpenStreetMap и 2GIS и обновляются автоматически.

**Принцип работы:** задайте вопрос так, как спросили бы коллегу — бот его поймёт.

> Бот не использует нейросети в рантайме. Вся логика — детерминированные правила на Python: keyword matching → планировщик операций → SQL к базе данных или HTTP к внешним API. Это значит: ответы воспроизводимы, предсказуемы и не «галлюцинируют».

## РАЗДЕЛ 1 — Роли пользователей
---

Перед первым использованием выберите роль — переключатель находится в **правом верхнем углу** рядом с кнопкой «API Docs». Роль определяет, какие подсказки и панели отображаются. Настройка сохраняется в браузере между сессиями.

### Смена роли голосом или текстом

| Что написать / сказать | Результат |
|---|---|
| `Сигма, я горожанин` | Режим жителя города |
| `Сигма, я служащий` | Режим сотрудника мэрии (весь город) |
| `Сигма, я служащий Советский` | Режим сотрудника с привязкой к Советскому району |
| `Сигма, я служащий Академгородок` | Режим сотрудника с привязкой к подрайону |
| `Сигма! Мэр` | Режим руководителя |

> Слово «Сигма» — wake-word для голосового ввода. В текстовом режиме его можно не писать. Знаки препинания после «Сигма» (запятая, точка, восклицательный знак) — допустимы, бот их игнорирует.

---

### 👤 Горожанин — режим по умолчанию

Для жителей города. Показывает повседневные подсказки: ЖКХ, транспорт, инфраструктура рядом с домом.

**Подсказки-чипы** в интерфейсе разделены на категории. Синие чипы со значком 📍 возвращают список объектов с адресами — под результатом появляется кнопка «Показать на карте 2GIS».

**Типичные запросы в режиме Горожанин:**

```
Есть горячая вода в Дзержинском районе?
Отключения электричества сегодня
Аптеки в Центральном районе
Школы в Академгородке
Библиотеки для детей
Как проехать из Академгородка в Дзержинский?
Пробки сейчас
Качество воздуха в городе
Риски для жизни в городе
```

### 🏙️ Служащий умного города

Для сотрудников мэрии и городских служб. При входе в режим автоматически загружается **панель диспетчера** — сводка по ключевым показателям:

| Плитка | Данные |
|---|---|
| 🔴 Аварийных домов | Сумма всех аварийных отключений ЖКХ |
| 🟡 Плановых домов | Запланированные отключения |
| 🍃 AQI средний | Средний индекс качества воздуха по всем районам |
| ⚠️ Активных рисков | Число активных угроз (экология, температура, гололёд) |

Под плитками — таблица топ-5 районов по авариям и кликабельные AQI-чипы по каждому району. Все четыре KPI-плитки кликабельны — открывают детализацию.

#### Фильтр по району ответственности

Служащий может закрепить свой район двумя способами:

**1. При выборе роли:** нажать «Служащий умного города ›» → выбрать район из пикера (10 районов + Академгородок + Кольцово + «Весь город»). Кнопка покажет: `🏙️ Советский`.

**2. Прямо в панели:** горизонтальная строка тегов под заголовком — клик по тегу мгновенно перефильтровывает все данные без перезагрузки страницы.

**Что фильтруется при выборе района:**
- KPI-плитки пересчитываются по данным конкретного района
- Таблица аварийных отключений — только строки этого района
- AQI-чипы — только чип выбранного района
- Все onclick-запросы автоматически добавляют контекст района

**Типичные запросы в режиме Служащий:**

```
Отключения электричества в Ленинском районе сегодня
Плановые отключения теплоснабжения на этой неделе
История отключений газа за 7 дней
Все коммунальные отключения сейчас
Активные стройки в Калининском районе
Стройки по районам
Ввод в эксплуатацию по районам
Сколько школ по районам
Камеры видеофиксации в Ленинском районе
```

---

### Индекс дорожной нагрузки (пробки)

Аналитический расчёт без GPS-трекеров — на основе исторических паттернов, праздничного календаря и погоды.

| Запрос | Что получите |
|---|---|
| `пробки сейчас` | Индекс 0–10, уровень, факторы нагрузки, совет |
| `трафик в городе` | То же |
| `загруженность дорог` | То же |
| `час пик сегодня` | Прогноз ближайшего пика |

#### Шкала индекса

| Значение | Уровень | Рекомендация |
|---|---|---|
| 0.0–1.9 | 🟢 Свободно | Отличное время для поездки |
| 2.0–3.4 | 🟡 Умеренно | Небольшие задержки возможны |
| 3.5–4.9 | 🟠 Затруднено | Заложите +15 минут |
| 5.0–6.4 | 🔴 Сложно | Рассмотрите общественный транспорт |
| 6.5–8.4 | 🔴 Очень сложно | Объезд / метро / самокат |
| 8.5–10.0 | ⛔ Коллапс | Останьтесь дома или используйте ОТ |

#### Факторы, которые учитываются

- Время суток и день недели (утренний/вечерний час пик)
- Официальные праздники и переносы РФ (база 2025–2027)
- Сезонные события: 1 Сентября (+2.0), канун Нового года (+2.0)
- Погода: снег (+1.0–2.5), гололёд (+2.5), ливень (+1.0), мороз ниже −20°C (+0.5)

## РАЗДЕЛ 3 — ЖКХ и коммунальные отключения
---

Данные обновляются каждые **~30 минут** с сайта 051.novo-sibirsk.ru. Бот отличает аварийные отключения от плановых.

### Запросы по отключениям

| Запрос | Операция |
|---|---|
| `все коммунальные отключения сейчас` | Все типы ресурсов: электро, тепло, вода, газ |
| `отключения электричества сегодня` | Только электроснабжение за сегодня |

| `история отключений газа за 7 дней` | Хроника по дням, до 7 дней назад |
| `отключения на этой неделе` | Сводка за текущую неделю |

### Как называть ресурсы

Бот понимает все бытовые варианты:

| Скажите | Бот ищет |
|---|---|
| «электричество», «свет», «электроэнергия», «электро» | Электроснабжение |

| «холодная вода», «вода», «ХВС» | Холодное водоснабжение |
| «газ» | Газоснабжение |
| «ЖКХ», «всё», «коммунальные», «ресурсы» | Все типы сразу |

### Как уточнить район

Добавьте район к любому запросу:

```

```

## РАЗДЕЛ 4 — Экология и погода
---

Данные от Open-Meteo — обновляются каждые **15 минут**. История хранится до 365 дней: скользящее окно 30 дней + постоянный архив за год.

### Запросы по экологии

| Запрос | Что получите |
|---|---|
| `качество воздуха в городе` | AQI, PM2.5, температура, ветер по всем 10 районам |
| `погода в Советском районе` | Данные по конкретному району |
| `погода в Академгородке` | Поддержка подрайонов |
| `превышение ПДК по PM2.5` | Районы с PM2.5 > 35 мкг/м³ (норма ВОЗ) |
| `динамика воздуха за неделю` | История по дням (до 30 дней) |
| `прогноз качества воздуха на завтра` | Прогноз: температура, ветер, AQI |
| `риски для жизни в городе` | Карточки активных рисков с рекомендациями |

### Карточки рисков — что означают

| Карточка | Триггер | Рекомендация |
|---|---|---|
| 🌫️ Экологическая ловушка | Ветер < 1.5 м/с + PM2.5 > 20 мкг/м³ | Горожанам: не открывайте окна. Службам: режим НМУ |
| ☢️ Превышение нормы ВОЗ | PM2.5 > 35 мкг/м³ | Горожанам: маска FFP2. Службам: оповещение населения |
| 🧊 Риск гололёда | Температура −3..+2°C | Горожанам: +20 мин на маршрут. Службам: пескоразбрасыватели |
| ⚠️ Температурный шок | Суточное падение ≥ 15°C | Горожанам: многослойная одежда. Службам: аварийный режим ЖКХ |
| 🥶 Экстремальный холод | Температура < −20°C | Горожанам: ограничьте время на улице. Службам: пункты обогрева |

### Индексы жизнедеятельности

Три расчётных индекса (0–10) на странице качества воздуха:

- **🚗 Индекс водителя** — опасность на дороге (гололёд, мороз, ветер)
- **🚶 Индекс прогулки** — комфорт на улице (PM2.5, AQI, температура)
- **🏗️ Коммунальное напряжение** — риск инцидентов ЖКХ (мороз + безветрие + смог)

## РАЗДЕЛ 8 — Карта 2GIS
---

Если в ответе есть список объектов (аптеки, школы, библиотеки и т.д.), появляется кнопка **«📍 Показать на карте»**. Нажмите — все точки отобразятся на интерактивной карте 2GIS. Клик по маркеру открывает попап с адресом, телефоном, режимом работы.

### Для каких запросов доступна карта

Карта работает для операций **FILTER** (список объектов с фильтром) и **TOP_N** (рейтинг). Синие чипы со значком 📍 в интерфейсе — визуальный маркер «этот запрос откроет карту».

| Запрос | Объекты на карте |
|---|---|
| `аптеки в Центральном районе` | Аптеки выбранного района |
| `топ-5 аптек` | До 5 точек |
| `библиотеки для детей` | Детские библиотеки |
| `школы в Советском районе` | Школы района |
| `детские спортивные организации` | Спортивные клубы и секции |
| `культурные организации` | Музеи, театры, центры культуры |
| `детские сады в Кировском районе` | Детские сады района |

**Карта не работает** для запросов `сколько...` (COUNT) и `по районам` (GROUP) — это агрегированные цифры без отдельных адресов.

### Как работает геокодирование

1. Нажимаете «📍 Показать на карте»
2. Бот геокодирует каждый адрес через 2GIS Geocoder API (~0.5–2 с на адрес)
3. Результат кешируется — повторные запросы мгновенны
4. На карте MapGL JS отображаются маркеры (максимум 50 объектов за запрос)

## РАЗДЕЛ 10 — Если бот не понял
---

Если запрос не распознан — бот не выдаёт ошибку, а показывает список доступных тем.

**Попробуйте:**

1. **Упростить фразу** — «школы Советский» вместо «образовательные учреждения в Советском районе»
2. **Использовать ключевые слова** — «отключения», «аптеки», «стройки», «пробки», «экология», «метро», «камеры»
3. **Задать другой формат** — вместо «покажи данные по паркингу» → «парковки по районам»
4. **Запросить справку** — напишите `что ты умеешь` — бот покажет все доступные возможности

**Примеры переформулировки:**

| Не распознаётся | Попробуйте так |

| «информация о транспортной сети» | «остановки по районам» |
