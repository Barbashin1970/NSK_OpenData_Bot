## РАЗДЕЛ 11 — Метро и аэропорт: детальные запросы
---

Разделы метро и аэропорта реализованы как самостоятельные модули со статическими данными, обогащёнными вручную. Это значит, что бот отвечает мгновенно — без обращения к внешним сервисам.

### 🚇 Метро Новосибирска

Новосибирский метрополитен — 2 линии, 13 станций, открыт в 1985 году. Среднесуточный пассажиропоток: **243 000 человек**. Стоимость проезда: **30 рублей**.

## РАЗДЕЛ 12 — IQ города: индексы и аналитика
---

СИГМА уже сегодня рассчитывает несколько **составных индексов** — числовых оценок состояния города. Это фундамент для полноценной концепции «IQ города» (City Intelligence Quotient).

### Что уже работает

| Индекс | Диапазон | Что измеряет |
|---|---|---|
| 🚗 **Индекс дорожной нагрузки** | 0–10 | Загруженность дорог: время, погода, праздники |
| 🌿 **AQI — индекс качества воздуха** | 0–5+ | PM2.5, PM10, NO₂ по 10 районам |
| 🚶 **Индекс прогулки** | 0–10 | Комфорт пешеходной среды |
| 🏗️ **Коммунальное напряжение** | 0–10 | Риск инцидентов ЖКХ |
| ⚠️ **Индекс активных рисков** | счётчик | Число одновременных угроз по экологии и ЖКХ |

**Запросить текущие индексы:**

```
пробки сейчас
качество воздуха в городе
риски для жизни в городе
коммунальное напряжение сегодня
```

### IQ города — концепция для следующей версии

**IQ города** — единый агрегированный показатель качества городской среды в реальном времени. Аналог: Barcelona Smart City Index, Moscow Urban Analytics.

Предлагаемая формула:

```
IQ_города = f(
  ЖКХ_стабильность,     # 1 − (аварийные_дома / норма)
  Воздух_качество,       # инверсия AQI
  Транспорт_плавность,   # 1 − (индекс_пробок / 10)
  Стройки_активность,    # динамика ввода в эксплуатацию
  Инфраструктура_доступность  # покрытие объектами на 1000 жителей
)
```

Итоговое значение: **0–100**, обновляется раз в 15 минут, отображается на дашборде мэра как единая плитка.

Ниже — предложения по развитию СИГМЫ, сгруппированные по приоритету и аудитории. Часть уже находится в разработке, часть — предложения для согласования с заказчиком.

### 🔴 P1 — Высокий приоритет (готово к реализации)

#### Расписание рейсов аэропорта
Интеграция с Яндекс.Расписания API (ключ уже упомянут в коде `airport_data.py`).

| Запрос | Функция |
|---|---|

| `когда прилетает рейс SU 1234` | Статус конкретного рейса |

#### История ЖКХ — расширенная аналитика
- «Сколько отключений было в Ленинском районе за последние 30 дней»

- «Динамика аварийных отключений по месяцам»

#### Пробки — реальный трафик
Подключение к Яндекс.Пробки API или 2GIS Traffic — замена аналитического индекса на данные GPS-трекеров.

### 🟠 P2 — Средний приоритет (для заказчика из Мэрии)

#### Паспорт района
Единая карточка по любому из 10 районов — «умная справка» для служащих и журналистов.

```
Советский район — паспорт
─────────────────────────
Население:       ~150 000
Школ:            18
Детских садов:   22
Аптек:           4
Парковок:        112 (1840 мест)
Активных строек: 7
AQI сейчас:      2.1 (умеренный)
Аварийных домов: 0
Текущие риски:   нет
```

**Запросы:**
```
паспорт Советского района
справка по Калининскому району
сводка по Дзержинскому
```

#### Сравнение районов
- «Сравни Советский и Центральный районы по экологии»
- «Где лучше воздух — в Академгородке или в Ленинском?»
- «Топ-3 района по числу школ на 1000 жителей»

#### Уведомления и подписки
Настройка оповещений для служащих мэрии:
- Push-уведомление при аварийном отключении в закреплённом районе
- Alert при превышении AQI > 3.5
- Утренний дайджест: сводка за ночь

#### Экспорт данных
- «Скачать список аптек в CSV»
- «Отчёт по стройкам Калининского района в PDF»
- Кнопка «📥 Скачать» под таблицами с результатами

### 🟡 P3 — Для презентации на ПМЭФ и конференциях

#### Демо-режим «Живой город»
Экран-презентация для стенда: автоматически обновляемая карта города с пятью слоями данных одновременно:
- Тепловая карта AQI по районам
- Точки аварийных отключений (красные маркеры)
- Камеры ПДД
- Активные стройки
- Индекс дорожной нагрузки в углу

#### IQ города — единый дашборд
Большой экран, 5 плиток + итоговый балл 0–100:

```
┌─────────────┬─────────────┬─────────────┐
│  🌿 Воздух  │  ⚡ ЖКХ     │  🚗 Трафик  │
│   AQI: 2.1  │  Норма 🟢   │  4.2 / 10   │
├─────────────┴──────┬──────┴─────────────┤
│  🏗️ Стройки       │  ⚠️ Риски          │
│  7 активных        │  0 угроз 🟢        │
├────────────────────┴────────────────────┤
│         🏆 IQ ГОРОДА: 74 / 100          │
│            ● Штатный режим              │
└─────────────────────────────────────────┘
```

#### Голосовой киоск
СИГМА как физический информационный терминал в МФЦ, администрации или аэропорту Толмачёво:
- Wake-word «СИГМА» → голосовой ввод вопроса
- Ответ озвучивается через TTS (Yandex SpeechKit или Silero)
- Экран показывает карту или таблицу
- Подходит для зон ожидания и стендов

#### Мультиязычность
Английский интерфейс для международных гостей ПМЭФ:
- `show metro stations` → English response
- `air quality today` → EN response
- `how to get to the airport` → EN response
