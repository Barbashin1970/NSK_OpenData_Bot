### Яндекс.Расписания API (приоритет)
- Эндпоинт: `https://api.rasp.yandex.net/v3.0/`
- Бесплатно без регистрации для non-commercial
- Коды НСК: станция `s9612870` (НСК-Главный), аэропорт `s9600370` (Толмачёво)
- Метро: код узла Новосибирского метро в базе Яндекса — поиск через `/search/?from=c66&transport_types=suburban`
- TTL: 30 мин
- Ключ: `YANDEX_RASP_KEY` (можно без ключа — лимит 1000 req/day)

## Технический стек расширений

### Новые источники данных

```
opendata.novo-sibirsk.ru/OpenDataDocuments/  — 63 набора, все в CSV
  Data52  → парки и скверы
  Data53  → детские площадки
  Data54  → освещение
  Data56  → МФЦ

data.gov.ru/opendata/  — федеральный уровень
  7711336883-medpomosh  → поликлиники и больницы (Минздрав)
  rosgvard-msk          → участки полиции

openbudget.mfnso.ru/opendata/  — Открытый бюджет НСО
  CSV по годам, кодировка UTF-8

Яндекс.Расписания API  → https://api.rasp.yandex.net/v3.0/
  Без ключа: 1000 req/day
  Ключ (бесплатно на developer.tech.yandex.ru): до 30 000 req/day
```

### Шаблон добавления нового топика (Quick reference)

1. **Данные**: добавить в `config/datasets.yaml` ИЛИ создать `src/<topic>_data.py`
2. **Роутер**: добавить `_route_<topic>()` + зарегистрировать в `route()` в `router.py`
3. **Планировщик**: добавить операцию в `Operation` (Literal) + ветку в `make_plan()` в `planner.py`
4. **Исполнитель**: добавить `execute_<topic>()` в `executor.py`
5. **API**: добавить ветку `if topic == "<topic>":` в `get_ask()` в `api.py`
6. **Рендер**: добавить `renderTopic()` + dispatch в `index.html`
